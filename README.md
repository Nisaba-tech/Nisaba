# OpenHam.radio

**RF + Code + Open Source = The Future of Amateur Radio**

[![Join Discord](https://img.shields.io/discord/1510784891782955008?style=for-the-badge&logo=discord&logoColor=white&label=Join%20Discord&color=5865F2)](https://discord.gg/Vb2Pfhb6J9)

## Vision

We are breaking the traditional "just casual QSO" model of amateur radio.  
We are building a **global, decentralized community** that combines RF technology with software development, hardware design, and maker culture.

---

## How to Join

1. **[Join our Discord Server](https://discord.gg/Vb2Pfhb6J9)**
2. Introduce yourself in `#introductions`
3. Verify your license in `#getting-licensed` (if you have one)
4. Explore and contribute on GitHub

---

## What We Offer

- **Multi-mode RF Bridge** (DMR main backbone)
- **Discord ↔ RF Bridge** (Web-to-RF)
- Weekly Tech Nets
- Open Source Projects & Hackathons

---

## Rules

- RF Transmission is **only allowed for licensed amateur radio operators**
- Unlicensed members can listen and participate in text channels
